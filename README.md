# StudyForge 🧠

> AI-powered student productivity OS — built for serious learners.

## Features
- 📝 AI Notes Summarizer
- 💬 PDF Chat
- ❓ AI Quiz Generator
- 📅 Assignment Planner
- 📊 Attendance Tracker
- 🎤 AI Viva Assistant

## Tech Stack
- **Frontend**: Next.js 15, Tailwind CSS, shadcn/ui
- **Backend**: Next.js API Routes, Supabase (PostgreSQL)
- **AI**: OpenAI API
- **Auth**: Clerk
- **Deployment**: Vercel

## Getting Started
\`\`\`bash
npm install
cp .env.example .env.local
# Add your API keys
npm run dev
\`\`\`
